# Specification

## Summary
**Goal:** Update the private event rental pricing in the PrivateEvents component to reflect new rates.

**Planned changes:**
- Update the 2-hour private event rental price from its current value to $650
- Update the 3-hour private event rental price from its current value to $800

**User-visible outcome:** Visitors viewing the Private Events section will see the updated prices of $650 for 2-hour rentals and $800 for 3-hour rentals.
